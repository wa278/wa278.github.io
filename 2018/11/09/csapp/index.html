计算机系统基础

信息（位与上下文）

像源文件这样由ASCII字符构成的文件称为文本文件，其他所有文件都称为二进制文件。

在不同的上下文中，一个同样的字节序列可能表示不同的信息。信息是由位（字节序列）和上下文决定的。

程序被翻译成不同的格式

以C语言为例 最初始 .c文件

  • 预处理阶段 读取 include<stdio.h>的内容，并且把它直接插入到程序文本中。 .i文件
  • 编译阶段 将文本文件翻译成汇编文件。 .s文件
  • 汇编阶段 将汇编文件翻译成 二进制文件 .o
  • 链接阶段 链接二进制文件（如链接printf函数存在的文件） 变成可执行目标文件

系统的硬件组成

  • 总线 传送定长的字节块，称为字。32位机4个字节，64位机8个字节。
  • IO设备 系统与外部联系的通道
  • 主存 线性的字节数组，每一个字节都具有唯一地址。
  • 处理器 处理器是解释存储在主存中指令的引擎，核心是一个字长的存储设备，即程序计数器。

高速缓存

高速缓存存放可能经常访问的数据，大部分存储其操作都能在快速的高速缓存中完成。

寄存器》L1缓存》L2缓存》L3缓存》主存》本地磁盘

操作系统管理硬件

所有应用程序对硬件的操作尝试必须通过操作系统。
几个基本抽象概念：

  • 文件 IO设备的抽象表示
  • 虚拟存储器 对主存和磁盘IO设备的抽象表示
  • 进程 对处理器，主存，IO设备的抽象表示。

进程

进程是操作系统提供的一种假象，好像系统上只有这个程序在运行。进程是操作系统对正在运行的程序的一种抽象。处理器在进程间进行切换，操作系统实现这种机制称为上下文切换。 操作系统保持跟踪进程运行需要的所有信息（上下文），包括程序计数器和寄存器和主存的内容。

线程

每个线程都运行在进程的上下文中，共享同样的代码和全局数据。

虚拟存储器

为进程提供了一种假象，即独占的使用主存。每个进程看到是一致的存储器，称为虚拟地址空间。（不同进程相同地址的虚拟地址空间会被映射到不同的物理地址上）

文件

系统中所有的输入输出都是通过使用一小组UNIX IO 的系统函数调用读写文件来实现的。

信息的处理和表示

字节顺序

最低有效字节在前面的方式，称为小端法。最高有效字节在最前面的方式称为大端法。不同的机器不一样。
网络应用程序代码的编写必须遵守已建立的关于字节顺序的规则。

表示字符串

在使用ASCII码作为字符串的任何系统上都将得到相同的结果，与字节顺序和字大小无关。（存的就是ASCII码），因此，文本数据比二进制数据具有更强的平台独立性。

表示代码

指令编码是不同的。不同机器类型使用不同且不兼容的指令和编码方式。同样机器运行不同的操作系统也会有不同的编码规则。

布尔代数

a^a = 0;
a^a^b = b;
0^b = b;

c语言中的逻辑运算

非0的参数都认为是True;0表示false。并且满足短路法则。

移位运算

逻辑移位 左移：丢弃最高的K位，右端补上k个0； 右移：左端补k个0，右端丢弃k位。
算术移位 左移：丢弃最高的K位，右端补上k个0； 右移：左端补k个最高位有效值，右端丢弃k位。

java全是有符号数，全部采用补码，所以是算术移位。如果移位超过字长，会采取求模的方式重新计算移位。

有符号数和无符号数转换

无符号数y = x + 2的W次方 x<0;
x x>=0;

有符号数y = x x < 2的w-1次方
x - 2W次方 x >= 2的w-1次方。

隐式转换 ：
-1 < 0U 因为0是无符号的，所以会将-1转化位无符号数42393967295u

拓展一个数字的位表示

  • 无符号数 零扩展 在最高为添加0；

  • 有符号数 符号扩展 添加最高位的副本。

  • 扩展与转化的顺序 ： short 转化位 unsigned 时，先改变大小，再完成转换。

截断数字

丢弃被截断的高位

无符号加法

计算的和 模 2的w次方，相当于简单丢弃w+1位。

补码加法

和无符号加法相同。计算机通过相同的机器指令完成。

无符号乘法

计算xy并 模 2的次方，相当于只保留w位。

补码乘法

和无符号乘法相同。计算机通过相同的机器指令完成。

乘以常数

用移位和加法运算来代替乘常数因子；
无符号数和补码都采用一下办法：x<< k 相当于 x 乘2的k次方。

x 乘 14 = (x<<3) + (x<<2) +(x<<1) 14 = 8 + 4 + 2

除以2的幂

对于无符号数 x >> k 等价于 x / 2的k次幂；

对于补码的非负数 x>>k即可，对于负数不需要舍入的情况，x >> k也可。
对于负数 需要舍入 如： -7/2 则需要计算 (x + (1<<k) -1)>>k，原因是负数需要像0舍入而不是向下舍入。
